## Sourceduty: Process_Theory

🔝 Linear activity sequences, trajectories, durations and intervals.

### USAGE

End-to-end diagram for software input and output.

### SYMBOLS

Multiprocess (⇄ ⇅ ⇆ ⇇ ⇈ ⇉ ⇊)
Preprocess (↝) 
Midprocess (↯) 
Postprocess (↦)
Reprocess (↺)
Alternating process(⇄)
Subprocesses (⇢)

Optional process (↷)
Joint process (↭)

### EXAMPLES

Process = 0 ↝ 1 → 2 ↷ (3 ↭ 4) → 5 ↺
Subprocess = 0 ⇢ 1 ⇢ 2 ⇢ 3
Multiprocess = 0/0 ↝ 1/1 ⇉ 2/2 ⇉ 3/3 ⇉ 4/4
